# PetersenLab Battlefield

A turn-based 1v1 strategy game born in the Petersen Lab.

## Overview

**PetersenLab Battlefield** is a browser-based duel game where you create a custom hero, allocate stats, and battle an opponent in real-time.

## Features

- **3 Archetypes**: Titan (Tank), Ghost (Speed), Maverick (Balanced).
- **Skill System**: Allocate 100 points across Chaos Strike, Vampire Soul, and Stun Lock.
- **Online Multiplayer**: Host a room and challenge a peer via direct link.
- **Visuals**: Dynamic particle effects and battle animations.

## Access

The application is secured. You must enter the lab key **"BRUS"** to access the battlefield.

## Tech Stack

- Vanilla JS + HTML5
- TailwindCSS
- PeerJS for WebRTC multiplayer
